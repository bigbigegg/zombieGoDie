# ZombieGoDie — 游戏设计文档

## 概述

玩家面对摄像头做手势，系统实时识别手势并触发对应攻击技能，消灭从屏幕右侧涌来的僵尸。

**平台**：桌面 Python  
**风格**：像素风（使用 CC0 免费素材）  
**操控**：单手手势识别

---

| 层 | 技术 | 版本要求 |

## 手势 → 攻击映射

| 手势 | 判定逻辑（关键点） | 攻击名称 | 效果 |
|---|---|---|---|
| 拳头 (Fist) | 所有4根手指弯曲 | 冲击波 | 近距离AOE，消灭屏幕左1/3范围内所有僵尸 |
| 食指指向 (Point) | 仅食指伸直 | 精准射击 | 单目标高伤害子弹，直线飞行 |
| 张开手掌 (Open) | 所有5根手指伸直 | 冰冻术 | 所有僵尸减速50%，持续3秒 |
| 剪刀手 (Peace) | 食指+中指伸直，其余弯曲 | 双线激光 | 两条水平激光，持续0.5秒 |
| 大拇指朝上 (Thumbs Up) | 仅拇指伸直，手竖向 | 投弹 | 落点AOE爆炸，延迟0.8秒引爆 |

### 手势判定规则

使用 MediaPipe 输出的21个手部关键点，核心逻辑：

- **手指伸直**：指尖Y坐标 < 第二关节Y坐标（图像坐标系，向下为正）
- **手指弯曲**：指尖Y坐标 > 第二关节Y坐标
- **防抖机制**：手势需连续 **5帧** 保持一致才触发攻击，避免误识别
- **冷却时间**：每种手势独立CD，范围1~3秒，防止连续刷技能

| 技能 | 冷却时间 |
|---|---|
| 冲击波 | 3秒 |
| 精准射击 | 1秒 |
| 冰冻术 | 5秒 |
| 双线激光 | 2秒 |
| 投弹 | 4秒 |

## 游戏机制

### 僵尸行为

- 从屏幕右侧随机Y坐标生成，向左匀速移动
- 到达左边界：玩家扣血，僵尸消失
- 死亡：播放像素风死亡动画，掉落分数浮字

### 僵尸类型

| 类型 | 血量 | 移速 | 得分 | 出现条件 |
|---|---|---|---|---|
| 普通僵尸 | 1 | 慢 | 10 | 全程 |
| 胖僵尸 | 3 | 很慢 | 30 | 第2波起 |
| 精英僵尸 | 8 | 中 | 80 | 每3波一次 |
| 快速僵尸 | 1 | 快 | 20 | 第4波起 |

### 难度递进

- 每 **30秒** 进入下一波
- 每波僵尸数量 +2，基础移速 ×1.1
- 第5波起同屏最多存在10只僵尸

### 得分系统

- 击杀普通僵尸：+10
- 连续击杀（2秒内）：Combo倍率加成（最高×3）
- 精英僵尸击杀：+80
- 生存时间奖励：每30秒 +50

### 玩家状态

- 初始血量：5 格
- 僵尸到达左边界：-1 格
- 血量归零：游戏结束，显示最终分数

## 配置参数（config.py 预览）

```python
# 窗口
SCREEN_W = 1280
SCREEN_H = 720
FPS = 60

# 摄像头小窗
CAM_W = 240
CAM_H = 180

# 手势
GESTURE_CONFIRM_FRAMES = 5   # 防抖：连续N帧确认

# 技能冷却（秒）
CD = {
    "fist":     3.0,
    "point":    1.0,
    "open":     5.0,
    "peace":    2.0,
    "thumbsup": 4.0,
}

# 难度
WAVE_INTERVAL = 30           # 秒
ZOMBIE_SPEED_SCALE = 1.1     # 每波速度倍率
ZOMBIE_COUNT_STEP = 2        # 每波增加僵尸数
```
