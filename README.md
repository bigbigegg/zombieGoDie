# ZombieGoDie 🧟‍♂️✊

**用手势消灭僵尸！** 面对摄像头做手势，实时触发攻击技能，击杀从屏幕右侧涌来的僵尸。

---

## 环境要求

- Python 3.9+
- macOS / Windows / Linux
- 摄像头（笔记本内置摄像头即可）

## 快速开始

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 运行游戏
python3 main.py
```

## 操作方法

| 手势 | 技能 | 效果 | 冷却 |
|---|---|---|---|
| ✊ 拳头 | 冲击波 | 清空屏幕左 1/3 区域内所有僵尸 | 3 秒 |
| ☝ 食指 | 精准射击 | 高速子弹直线飞行，单体高伤害 | 1 秒 |
| ✋ 手掌 | 冰冻术 | 全屏僵尸减速 50%，持续 3 秒 | 5 秒 |
| ✌ 剪刀手 | 双线激光 | 两条水平激光持续 0.5 秒 | 2 秒 |
| 👍 大拇指 | 投弹 | 延时 0.8 秒范围爆炸 AOE | 4 秒 |

**按键说明**

| 按键 | 功能 |
|---|---|
| `空格` | 开场界面开始游戏 |
| `D` | 切换调试侧边栏（显示摄像头+手势信息） |
| `R` | 游戏结束后重新开始 |
| `ESC` | 退出游戏 |

## 游戏机制

- **僵尸**：从右侧生成向左移动，到达左边界扣血
- **波次**：每 30 秒一波，僵尸越来越多、越来越快
- **连击**：2 秒内连续击杀触发倍率加成（最高 ×3）
- **血量**：共 5 格，归零则游戏结束

## 调试模式

按 `D` 键打开左侧调试面板，实时显示：

- 摄像头画面 + MediaPipe 骨骼线
- 当前手势识别结果
- 防抖确认手势
- 五根手指的伸/曲状态
- 帧率

## 项目结构

```
zombieGoDie/
├── main.py                # 游戏入口 + 调试面板
├── config.py              # 全局配置 + 中文字体加载
├── requirements.txt       # Python 依赖
│
├── gesture/               # 手势识别模块
│   ├── detector.py        # MediaPipe 封装
│   └── classifier.py      # 关键点 → 手势判定 + 防抖
│
├── game/                  # 游戏逻辑
│   ├── scene.py           # 主场景（开场/波次/游戏结束）
│   ├── zombie.py          # 4 种僵尸实体
│   ├── attack.py          # 5 种攻击技能
│   ├── player.py          # 玩家状态（血量/得分/CD）
│   └── hud.py             # UI 渲染（状态栏/技能栏）
│
├── assets/                # 素材目录（待填充像素图）
│   ├── sprites/
│   └── sounds/
│
└── DESIGN.md              # 完整设计文档
```
